    Safiya H Discuss about the different studied algorithm design techniques.
  • Kruskal's Algorithm: Greedy approach which chooses the smallest edges in non-decreasing order, so no cycles are formed. Best for sparse graphs (few edges).
  • Prim's Algorithm: Greedy approach that grows the spanning tree by choosing the smallest edge connecting a vertex in the tree to one not in the tree. Best for dense graphs (many edges).
  • Dijkstra's Algorithm: Greedy approach that computes the shortest path from a source to all other nodes. Works best for graphs with non-negative weights.
  • Bellman-Ford Algorithm: Iterative approach that computes shortest paths, even with negative edge weights. The value really comes when detecting negative weight cycles.
  • Floyd-Warshall Algorithm: DP solution to find shortest paths between all pairs of nodes. Effective when dealing with small to medium-sized dense graphs.
