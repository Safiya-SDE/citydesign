🛠️ Course Introduction

  • Design Analysis and Algorithms: The course Design Analysis and Algorithms is the study of algorithms to solve, design, and implement for us to get an efficient solution.   • Data Structures and Algorithms currently studying:
    • Tracing, Hashing
    • DFS, BFS

    • Sorting Algorithm: Kruskal's, Dijkstra's, Prim's, Floyd's Algorithm
  • Real-Time Applications: Design Analysis and Algorithms: Algorithms are nothing but a simple breakdown of steps, wherein we use them to make our decisions efficient, faster, and smarter.
  • Some scenarios are: Finding the fastest way home, Using Zomato and Swiggy so that you'll get the order delivered from a nearby place with fast delivery and the plus point being offers.
  • Software Applications: Gaming, Socializing Apps, E-Commerce Platforms, Google Maps, etc.
